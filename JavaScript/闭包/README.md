# 闭包

```JavaScript
function f1() {
  var a = 1;

　function f2() {
　　console.log(a);
　}

　return f2;
}

var f = f1();

f(); // 1
```

上面的 `f2` 函数，就是**闭包**。

闭包就是能够读取其他函数内部变量的函数。

由于在 JavaScript 的语言中，只有函数内部的子函数才能读取局部变量，因此可以把闭包简单的理解为：定义在一个函数内部的函数。

- 闭包可以读取函数的内部变量，所以可以用来实现类（把父函数当作对象使用，把闭包当作公有成员函数，把内部变量当作私有成员变量）。
- 闭包可以让函数的内部变量始终保持在内存里，并且不会被垃圾回收机制回收（函数中的内部变量如果没有被任何闭包所引用，则会被回收）。
